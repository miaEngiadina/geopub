GeoPub

GeoPub is a small demonstrator showing how the ActivityPub protocol can be used to decentrally publish structured data with geolocation.

Let's start by logging in to an ActivityPub server.

You will need to register on a Pleroma instance before logging in. You can find a list of instances here.

On the right hand side of the screen you can see a map. By clicking on the map you can create a new note at the specific location.

Existing notes will be displayed on the map with a marker. Click on a marker to see the message

The notes are stored on the ActivityPub server (the Pleroma instance) and you can also see them from any other interface to Pleroma. You (and anybody who is following you) can even like or comment a note you create here.

Not only your own notes are dispayed on the map. All notes with geolocation in your inbox are displayed on the map. Try following another user who is creating notes with geolocation!

For more information see the repository.
